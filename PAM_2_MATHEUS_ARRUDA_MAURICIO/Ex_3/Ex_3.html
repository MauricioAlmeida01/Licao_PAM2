3) De exemplo dos seguintes formas de trabalhar com DOM(Document Object Model), e explique:

document.getElementById()

O "document.getElementById()" retorna o elemento cujo atributo ID foi especificado. Caso o elemento não exista ou o ID informado seja incorreto ele retornará null. Mas nesse exemplo pegarei o conteúdo da caixa de texto e mostrarei um alerta para o usuário com o texto que ele escreveu, se apertar o botão acima.





document.getElementsByTagName()

O "document.getElementByTagName()" retorna uma coleção de objetos através das tags html. Nesse exemplo mostra quantos elementos "p" tem no documento.





document.getElementsByClassName()

O "document.getElementsByClassName()" retorna um vetor de objetos com todos os elementos filhos que possuem o nome da classe dada. Se clicar no botão vai mostrar quantos elementos html com a classe "classes" tem no documento.





document.querySelector()

O "document.querySelector()" busca elementos no DOM com base num seletor CSS. Neste exemplo se clicar no botão, depois clicar com o botão direito do mouse, for em "inspecionar" e for na parte console verá o primeiro elemento que tem a classe exemplos, sendo que tem dois (marcados em azul).





document.querySelectorAll();

O "document.querySelectorAll()" Retorna uma lista de elementos presentes no documento (usando ordenação em profundidade, pré-ordenada e transversal dos nós do documento) que coincidam com o grupo de seletores especificado. Se o usuário clicar no botão mostrará todos elementos paragrafos ("p") do HTML e também elementos com a classe ("exemplos") no console.

Dica: para ir no console clique com o botão direito, vá em "Inspecionar" e depois clique na aba Console para ver o resultado
